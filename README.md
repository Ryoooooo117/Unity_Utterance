# Unity_Utterance
Edit the data in Assets/StreamingAssets/data.json to set utterance objects

example:
{
  "utteranceObjects":[
        {
            "name":"boy",
            "tags":[
                "Boy",
                "Nick",
                "He"
            ],
            "resource":"Prefabs/Boy_Prefab",
            "position":[0,0,0],
            "rotation":[0,0,0],
            "offsetPosition":[0,0,0],
            "priority":1
        },
        ...
  ]
}
